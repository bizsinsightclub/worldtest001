# tonemanner.md — 번역 / 어투 정립

위키 primary 언어는 **한국어**. 영문 원본(`lorebook_export.json`)을 자연스러운 한국어로 옮긴다. 기준은 **im-not-ai 가이드라인**(https://github.com/epoko77-ai/im-not-ai) — AI 번역투를 지운다.

## 핵심 원칙

- 입말에 가까운 자연스러운 한국어. 원문의 장르·톤 유지(설정문은 설정문답게, 대사는 대사답게).
- 사실·숫자·고유명사·**인용문(따옴표 안)**·대사 예시는 **원문 그대로**. 특히 로어북의 한국어 대사 예시는 손대지 않는다.
- 의역하되 정보 누락 금지. 과한 재창작 금지(체감 수정률 30% 이하 지향).

## 금지 (AI 시그니처)

| 금지 | 교정 |
|---|---|
| `~를 통해` | `~로`, `~으로` |
| `~에 의해` | `~가`(능동) |
| `~에 있어서`, `~에 대해` | 삭제 또는 재구성 |
| 이중 피동 `~되어진다`, `~지게 된다` | 단순 능동/피동 |
| 기계적 동일 어미 반복(`~이다.~이다.~이다.`) | 어미·문장 길이 변주 |
| 헤지 `~할 수 있을 것으로 보인다` | 단정하거나 근거 명시 |
| 상투어 `결론적으로`, `주목할 만하다`, `시사하는 바가 크다` | 삭제 |
| 문두 남발 `또한/따라서/즉` | 줄이기 |
| 불필요한 괄호 영문 병기 | 용어집 외에는 생략 |
| 과한 볼드·세미콜론·대시·이모지 | 절제 |

## 이름 · 표기

- **일본식 성-이름 순** 유지(예: 미카미 아카네 — 아카네 미카미 아님).
- 인물 이름: 한글이 메인, 필요 시 `<ruby>`로 한자·로마자 보조. 스테이지명도 동일(예: 아카카게로우 / 赤陽炎·Akakagerō).
- 관계 목록의 대상 이름은 **알려진 캐릭터면 한국어 정식명**으로(빌드의 관계 해석·이미지 매칭을 위해).

## 캐릭터 말투

각 프로필의 `Speech style` 섹션과 대사 예시가 캐릭터 톤을 규정한다. 번역 시 **존댓말/반말·방언·1인칭·말버릇**을 보존한다(예: 미카미 아카네 = 격조 있는 존댓말, 교토 야구 얘기엔 흥분). 설정 산문은 중립적 설명체, 대사·예시는 캐릭터 보이스 그대로.

## 용어 표기 규칙 (한글 메인 + 일어/영문 병기)

주 언어는 **한글**, 배경은 일본이다. 영문을 앞세우지 말고, 괄호 안에 **일어(한자+후리가나) 또는 영문**을 병기한다. `한글(똑같은 한글)` 중복은 금지.

- 형식: `한글 ( Romaji / 漢字(ふりがな) )` — 정의·헤더·첫 등장에 적용. 이후 반복 등장은 한글만.
- 잘못: `Magical Girl Management Offices`(영문 헤더), `마법소녀 사무소(마법소녀 사무소)`(중복).
- 올바름: `마법소녀 사무소 ( Mahou Shojo Office / 魔法少女事務所(まほうしょうじょ じむしょ) )`.

## 전투 장면 작법 (Battle Narration)

전투 예시 샘플은 원작 "전투 추가 가이드라인"을 따른다. 핵심:

- **전술적 주고받기**로 묘사한다. 명확한 전술적 이유 없이 즉승·즉패·굴복·경외·공포를 만들지 않는다. 강자도 저항·적응·반격·실수·대가·전장 변화에 부딪힌다.
- 적을 구경꾼으로 만들지 않는다. 상대는 자기 **등급·성격·전투방식·목적·공포내성·가용 능력**대로 움직인다. 압박·부상·후퇴는 있어도 강하다는 이유로 알아서 무너지지 않는다.
- 막연한 찬사("압도적이었다", "차원이 다르다") 대신 **구체적 전술 결과**를 쓴다. 예: "결계가 첫 일격을 흡수했지만, 적은 각도를 바꿔 두 걸음 물러서게 만들었다."
- 고유능력은 시그니처일 뿐 유일 수단이 아니다. **공통마법(결계/기동/디버프/자기강화)·무기·기동·지형·팀워크**를 조합한다. 특화에 맞게: 결계형은 막고 지연·보호, 기동형은 회피·재배치·빈틈 창출, 디버프형은 약화·둔화·교란·속박, 자기강화형은 속도·근력·반사·근접 압박.
- 권력 차·기습·준비·맥락이 명백하지 않으면 **여러 합** 끝에 결판. 강자의 우세는 **판단·위치·타이밍·자원 관리·전술적 압박**으로 보여준다.
- **캐릭터 일관성**: 각자의 전투 스타일·성격·말투(존댓말/반말·방언·1인칭)를 살린다. 대사는 인물 보이스 그대로, 서술은 절제된 한국어 산문(번역투 금지, im-not-ai 준수). 분량 1000~1500자.

## 빌드 연동 규칙

번역 산출(`lorebook_ko.json`)은 원문의 **`- English Label:` 필드 라벨과 `#### Section` 헤더를 영문 그대로 유지**하고 값·산문만 한국어로 옮긴다(파서 호환). 표시 단계에서 라벨을 한국어로 매핑한다.
