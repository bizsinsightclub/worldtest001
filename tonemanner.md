# tonemanner.md — 번역 / 어투 정립

위키 primary 언어는 **한국어**. 영문 원본(`lorebook_export.json`)을 자연스러운 한국어로 옮긴다. 기준은 **im-not-ai 가이드라인**(https://github.com/epoko77-ai/im-not-ai) — AI 번역투를 지운다.

## 핵심 원칙

- 입말에 가까운 자연스러운 한국어. 원문의 장르·톤 유지(설정문은 설정문답게, 대사는 대사답게).
- 사실·숫자·고유명사·**인용문(따옴표 안)**·대사 예시는 **원문 그대로**. 특히 로어북의 한국어 대사 예시는 손대지 않는다.
- 의역하되 정보 누락 금지. 과한 재창작 금지(체감 수정률 30% 이하 지향).

## 금지 (AI 시그니처)

| 금지 | 교정 |
|---|---|
| `~를 통해` | `~로`, `~으로` |
| `~에 의해` | `~가`(능동) |
| `~에 있어서`, `~에 대해` | 삭제 또는 재구성 |
| 이중 피동 `~되어진다`, `~지게 된다` | 단순 능동/피동 |
| 기계적 동일 어미 반복(`~이다.~이다.~이다.`) | 어미·문장 길이 변주 |
| 헤지 `~할 수 있을 것으로 보인다` | 단정하거나 근거 명시 |
| 상투어 `결론적으로`, `주목할 만하다`, `시사하는 바가 크다` | 삭제 |
| 문두 남발 `또한/따라서/즉` | 줄이기 |
| 불필요한 괄호 영문 병기 | 용어집 외에는 생략 |
| 과한 볼드·세미콜론·대시·이모지 | 절제 |

### 번역체 문장 사슬 (산문·서사에서 특히 금지)

영어식 대조 구문을 한국어로 직역한 패턴들. 한두 번은 몰라도 **반복하면 번역투·분석조**가 된다.

| 금지 패턴 | 비고 |
|---|---|
| `단순한 ~가 아니었다` | "이것은 단순한 소리가 아니었다" 류 — 설명조. |
| `A가 아니었다. B였다.` | 부정→정정 2단 구문 반복 금지. |
| `더 이상 ~가 아니었다` | "Jane의 째깍거림은 더 이상 평소의 소리가 아니었다" 류. |
| `~일 뿐이었다` | 감정을 설명으로 깎아내림. |
| `그저`, `단지` 남발 | 감정 요약용으로 자주 쓰지 말 것. |
| 메타 복선 `이것은 시작에 불과했다`, `그들은 아직 몰랐다` | 미래 암시 금지 — 현재에 집중. |

## 이름 · 표기

- **일본식 성-이름 순** 유지(예: 미카미 아카네 — 아카네 미카미 아님).
- 인물 이름: 한글이 메인, 필요 시 `<ruby>`로 한자·로마자 보조. 스테이지명도 동일(예: 아카카게로우 / 赤陽炎·Akakagerō).
- 관계 목록의 대상 이름은 **알려진 캐릭터면 한국어 정식명**으로(빌드의 관계 해석·이미지 매칭을 위해).

## 캐릭터 말투

각 프로필의 `Speech style` 섹션과 대사 예시가 캐릭터 톤을 규정한다. 번역 시 **존댓말/반말·방언·1인칭·말버릇**을 보존한다(예: 미카미 아카네 = 격조 있는 존댓말, 교토 야구 얘기엔 흥분). 설정 산문은 중립적 설명체, 대사·예시는 캐릭터 보이스 그대로.

## 빌드 연동 규칙

번역 산출(`lorebook_ko.json`)은 원문의 **`- English Label:` 필드 라벨과 `#### Section` 헤더를 영문 그대로 유지**하고 값·산문만 한국어로 옮긴다(파서 호환). 표시 단계에서 라벨을 한국어로 매핑한다.

## 플레이 앱 실시간 서사 문체

플레이 앱(`lorebook-play.html`)은 매 턴 LLM에 시스템 프롬프트를 보내 라이트노벨 서사를 실시간 생성한다. 그 문체 디렉티브는 `play_template.py`의 `assembleContext()`가 담당하며(런타임 핵심), 본 문서가 정전 기준이다. 핵심 규칙 요지:

- **서술 = 현재형 평어체**(`~다/~ㄴ다/~았다·었다`). 격식체(`~합니다`)는 그렇게 말하는 캐릭터의 **대사 안**에서만. 무미건조한 보고형 묘사 금지 — 시점 인물의 감각·태도가 묻어나게.
- **말하지 말고 보여줘라**: "놀랐다/긴장했다" 대신 반응(숨 멈춤·시선 회피·손끝 떨림). 오감·미세 몸짓·짧은 내면 독백, 의성어·의태어. 정보는 환경·대사·행동에 녹여(인포덤프 금지). 한 번 묘사한 건 변하지 않는 한 재묘사하지 않음.
- **대사가 이야기를 끌고 간다**(묘사보다 비중↑). 캐논 말투(존댓말/반말·방언·1인칭·말버릇) 끝까지 보존. 짧고 자연스러운 리듬·말줄임·침묵. 관계는 말투로 드러냄.
- **금지**: 위 「번역체 문장 사슬」 전부 + 메타 복선 + 직전 응답 문장·표현 반복 + 애니풍 과장·주인공 버프.
- **인물**: 프로필은 작업지시서가 아니다 — 시트 밖 행동 허용, 체크리스트 연기 금지. 성격은 행동·대사로.
- **시점·진행**: 2인칭("당신"). 시점 인물이 모르는 건 서술도 모름. 2~4문단. 점진적·상호작용형 전개, 메타 질문/선택지 나열로 턴을 닫지 말 것.
- **표기**: 한국어 우선, 고유명사·기술명은 첫 등장만 괄호 병기.
