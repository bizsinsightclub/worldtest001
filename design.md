# design.md — 디자인 룰

마법소녀 + 괴이 호러 정서의 **빈티지·다크** 위키. 레퍼런스: **Reverse:1999** 캐릭터 컬렉션 UI(세피아·황동·오너먼트 프레임·레어리티 별·분위기 있는 인물 일러스트). Claude 기본 톤을 넘어 세계관에 맞춘다.

## 팔레트 (CSS 토큰)

```
--bg:#16110b  --bg2:#1f1810  --panel:#241c12  --panel2:#2c2216
--ink:#ece0c8  --ink-dim:#b6a784  --ink-faint:#8a7a5c
--gold:#c9a24b  --gold-bright:#e6c879  --gold-deep:#8a6d2c
--line:#3d3019  --line-soft:#2b2114
```

배경은 따뜻한 차콜/세피아 + 종이 질감 노이즈(인라인 SVG) + 비네팅. 골드는 강조·테두리·선택 상태에만.

### 팩션 7색 (서로 충분히 구분되도록)
| 팩션 | 토큰 | 색 의도 |
|---|---|---|
| 지역 대표 rep | `--rep` | 골드/황동 |
| 마법소녀청·협력 bureau | `--bureau` | 청록 |
| 연습생 trainee | `--trainee` | 연녹 |
| 시고쿠(타마모) shikoku | `--shikoku` | 여우빛 자홍 |
| 졸업거부회 refusal | `--refusal` | 적동(붉은 기) |
| 괴이·빌런 villain | `--villain` | 보라 |
| 게스트 guest | `--guest` | 회색 |

팩션색은 **칩·필터·도감 카드 테두리/리본·관계도 노드 링**에 일관 적용해 그룹이 한눈에 보이게 한다. 색만으로 구분이 약하면 명도·채도를 벌린다.

## 타이포 · 루비

- 디스플레이(제목): `'Iowan Old Style','Palatino Linotype','Book Antiqua',Palatino,'Nanum Myeongjo',Georgia,serif`.
- 본문: `'Pretendard','Apple SD Gothic Neo','Malgun Gothic',system-ui,sans-serif`.
- **루비 표기**: 한글이 메인, 한자/로마자는 `<ruby><rt>`로 작게 위첨자. 스테이지명·고유명사·개념어에만. `rt`는 `--gold`, 0.5em. 본문에 영문을 길게 노출하지 않는다(→ tonemanner).
- 보조 영문(이름 아래 `日本語 · Romaji`)은 `.sub` 클래스로 작게·이탤릭·`--ink-faint`.

## 컴포넌트 패턴

- **도감 카드**: 3:4 인물 썸네일, 오너먼트 테두리, 팩션색 리본/테두리, 랭크 별(★), 한글명 + 작은 영문. hover 시 부상·골드 테두리.
- **상세 패널**: 대형 일러스트 + 분위기 배경, 별 랭크, 이름(루비), 팩션 칩, 스탯 그리드, 기본↔변신 토글, "상세 설명" → 모달(전체 프로필 마크다운).
- **팩션 칩**: 크게(패딩·폰트↑), 반투명 팩션색 배경 + 진한 팩션색 테두리 + 팩션색 텍스트, 좌측 도트/엠블럼.
- **스탯 그리드**: 2열, 라벨(작은 caps·`--ink-faint`) + 값(`--ink`), 대비·크기 충분히. 팩션색 액센트 라인 허용.
- **필터 버튼**: 각 팩션 고유색 틴트, 활성 시 팩션색 채움.
- **모달**: 다크 패널 + 골드-deep 테두리, 마크다운 렌더(h2 골드, 불릿 `--ink-dim`).

## 모션

- hover: 0.14s transform/box-shadow.
- **연표 각인**: 마일스톤이 뷰포트 진입 시 "그을음/불에 새겨지듯" 리빌 — ember 그라데이션 + 마스크 스윕 → 식으면 각인된 골드 텍스트. 가운데 세로선은 타들어가는 도화선 느낌.
- 접근성: `prefers-reduced-motion` 시 단순 페이드로 대체.

## 지도 · 그래프 테마

- **지도**: 어두운 해양 배경, 도도부현 실측 경계(가는 스트로크) + 구역별 채색(팩션색 아님, 지역 식별용 무채/세피아 계열), 호버/선택은 골드, 시고쿠는 봉인(암적색·점선·⛩).
- **관계도**: 팩션색 노드 링, 팩션별 군집, 관계 유형별 엣지 색(친구/라이벌/스승/가족/적대/연심/동료). 범례 상시.
